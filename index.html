Standalone Mobile App Setup

By default, this collaborative sandbox is nested within an iframe that forces headers and safety banners. Follow these two simple steps to run it as a completely native, borderless full-screen application on your phone!

1 Copy Code Base

Grab the compiled single-file codebase containing all styling, logical charts, and database setups.

2 Free Dropping & Launcher Creation

  • Create a plain-text file named index.html on your desktop and paste the copied code into it.
  • Drag and drop that file onto Netlify Drop or upload to Tiny.host for immediate free hosting.
  • Open your new custom Netlify link on your iPhone Safari or Android Chrome browser.
  • Choose Add to Home Screen. It will now open as a pristine, immersive full-screen application!

*Note: Your Firestore credentials and existing daily logs are fully synchronized via the cloud, meaning you won't lose any records when deploying!
